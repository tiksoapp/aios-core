# AIOS 代理角色定义

> 🌐 [EN](../../agents/persona-definitions.md) | [PT](../../pt/agents/persona-definitions.md) | [ES](../../es/agents/persona-definitions.md) | **ZH**

**版本:** 1.0
**创建日期:** 2025-01-14
**作者:** @ux-design-expert (Uma)
**目的:** 为所有12个AIOS代理提供完整的角色定义，以实现具有一致身份的命名代理系统

---

## 🎯 概述

本文档为12个AIOS代理定义了完整的角色，包括：

- **名称:** 性别中立、全球可发音（EN + PT-BR）
- **职责:** 清晰的功能描述
- **原型:** 基于星座的个性关联
- **颜色:** 6色无障碍测试调色板
- **图标:** 可识别的表情符号
- **特征:** 个性特点
- **沟通风格:** 交互模式
- **原型行为:** 三级问候行为

**注意:** 原计划13个代理，但根据决策文档 `docs/decisions/security-agent-vs-security-module-decision.md`，@security (Apex) 已被取消。安全功能将作为横切模块实现。

---

## 🌈 调色板

6色调色板设计原则：

- **无障碍性:** 符合WCAG AA标准（最小对比度4.5:1）
- **功能分组:** 相似代理类型使用相关颜色
- **视觉区分:** 一目了然识别代理

| 颜色       | 十六进制  | 用途                           | WCAG AA |
| ---------- | --------- | ------------------------------ | ------- |
| **青色**   | `#00BCD4` | 创新、流程、元编排             | ✅ 通过 |
| **绿色**   | `#4CAF50` | 质量、自动化、培育             | ✅ 通过 |
| **黄色**   | `#FFC107` | 平衡、分析、数据               | ✅ 通过 |
| **红色**   | `#F44336` | 调查、关键分析                 | ✅ 通过 |
| **灰色**   | `#607D8B` | 战略、规划、结构               | ✅ 通过 |
| **品红色** | `#E91E63` | 愿景、架构、创意               | ✅ 通过 |
| **蓝色**   | `#2196F3` | 文档、内容、知识               | ✅ 通过 |

## 🎨 设计原理摘要

### 名称选择标准:

1. **性别中立** - 在全球范围内无性别关联
2. **简短易记** - 3-6个字符，易于输入和记忆
3. **可发音** - EN和PT-BR中发音清晰
4. **有意义** - 每个名称与角色有语义联系
5. **专业** - 适合企业环境

### 原型映射理念:

- **基于角色的对齐** - 原型与代理功能匹配
- **能量共鸣** - 个性特征与星座特征一致
- **文化普遍性** - 星座在全球被认可
- **无刻板印象** - 原型是期望性的，而非限制性的

### 颜色系统设计:

- **无障碍优先** - 所有颜色符合WCAG AA标准
- **功能分组** - 相关代理共享颜色系列
- **视觉层次** - 易于扫描和识别代理
- **专业调色板** - 受Material Design启发，现代而永恒

### 图标选择:

- **基于表情符号** - 通用、到处渲染、无障碍
- **语义清晰** - 图标直接代表功能
- **视觉区分** - 每个图标独特且可识别
- **可缩放** - 从CLI到GUI任何尺寸都适用
