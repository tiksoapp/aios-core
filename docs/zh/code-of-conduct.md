# 贡献者契约行为准则

> 🇧🇷 [葡萄牙語版本](../CODE_OF_CONDUCT-PT.md)

## 我们的承诺

作为成员、贡献者和领导者，我们承诺使我们社区的参与是一个
不受骚扰的体验，无论年龄、体型、可见或不可见的残疾、种族、
性别特征、性别认同和表达、经验水平、教育、社会经济状况、
国籍、个人外表、种族、种姓、肤色、宗教或性身份和性取向。

我们承诺以有助于开放、热情、
多样化、包容和健康社区的方式行动和互动。

## 我们的标准

有助于我们社区创建积极环境的行为示例包括：

* 使用欢迎和包容的语言
* 尊重不同的观点和经历
* 优雅地接受建设性批评
* 专注于对社区最有利的事情
* 对其他社区成员表达同情

不可接受的行为示例包括：

* 使用性化的语言或意象，以及任何形式的性关注或性骚扰
* 灵活的骂名、侮辱或贬损的评论，以及个人或政治攻击
* 公开或私下骚扰
* 未经明确许可发布他人的私人信息，如物理地址或电子邮件地址
* 其他在专业环境中可以合理认为是不适当的行为

## 执法职责

社区领导者负责澄清和执行我们可接受行为的标准，并将
采取适当和公平的纠正措施，以应对他们认为不适当、令人
吓唬、令人反感或有害的任何行为。

社区领导者有权利和责任删除、编辑或拒绝
评论、提交、代码、wiki 编辑、问题和其他不符合本行为
准则的贡献，并将在适当时交流审核决定的原因。

## 范围

本行为准则适用于所有社区空间内，也适用于
当个人在公共空间中正式代表社区时。代表我们社区的
示例包括使用官方电子邮件地址、通过官方社交媒体账户发布，
或在在线或离线活动中充当任命的代表。

## 执法

可能向负责执法的社区领导者报告不当、骚扰或
其他不可接受的行为实例，电子邮件地址为
conduct@SynkraAI.com。所有投诉将被迅速和公平地审查和调查。

所有社区领导者有义务尊重任何事件报告者的隐私和安全。

## 执法指南

社区领导者在确定任何违反本行为准则的行为的后果时
将遵循这些社区影响指南：

### 1. 纠正

**社区影响**：使用不适当的语言或其他行为被认为
是不专业的或在社区中不受欢迎的。

**后果**：来自社区领导者的私人书面警告，
澄清违规的性质并解释为什么该行为是不适当的。
可能会要求提出公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列行动的违规。

**后果**：警告后果持续行为。在指定的时间段内，
不与涉及的人进行互动，包括与执行行为准则的人的无意
互动。这包括避免在社区空间以及社交媒体等外部渠道中的互动。
违反这些条款可能导致临时或永久禁止。

### 3. 临时禁止

**社区影响**：严重违反社区标准，包括
持续的不适当行为。

**后果**：在指定的时间段内，禁止与社区进行任何形式的
互动或公开交流。在此期间，不允许与涉及的人进行公开或
私人互动，包括与执行行为准则的人的无意互动。
违反这些条款可能导致永久禁止。

### 4. 永久禁止

**社区影响**：展示违反社区标准的行为模式，
包括持续的不适当行为、骚扰个人或侵略或
诋毁各类个人。

**后果**：永久禁止在社区内进行任何形式的公开互动。
