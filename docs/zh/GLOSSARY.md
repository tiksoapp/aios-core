## 翻译风格指南

### 1. 技术文档风格
- 使用正式、专业的语言
- 避免口语化表达
- 保持句子简洁

### 2. 代码相关内容
```markdown
# 正确
使用 `npm install` 安装依赖

# 错误
使用 `npm 安装` 来安装依赖
```

### 3. 命令和路径
```markdown
# 正确
运行 `npx aios-core init` 初始化项目

# 错误
运行 `npx aios-core 初始化` 初始化项目
```

### 4. 变量和参数
```markdown
# 正确
设置 `AIOS_DEBUG=true` 启用调试模式

# 错误
设置 `AIOS_调试=真` 启用调试模式
```
